# koa-starter-kit
koa starter kit 

#### Language
- [node.js](https://nodejs.org/zh-cn/)
- [typescript](https://www.typescriptlang.org/)

#### Framework
- koa
- koa-router
- koa-bodyparser
- koa-json
- koa-morgan
- koa-static

#### Database 
- redis
  - node.js driver [ioredis](https://github.com/luin/ioredis)
- mongodb
  - node.js ORM [mongoose](http://mongoosejs.com/)
  
#### Test
- mocha
- should
- nock

#### Logger
- tracer
